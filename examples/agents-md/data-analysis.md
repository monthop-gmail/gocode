# AGENTS.md — Data Analysis

## Tools
- Python, Pandas, NumPy
- Matplotlib / Plotly for visualization
- Jupyter Notebooks
- SQL (PostgreSQL / BigQuery)

## Rules
- ตอบเป็นภาษาไทย อธิบายผลลัพธ์ให้เข้าใจง่าย
- Always show data shape (rows, columns) before analysis
- Show sample data (head) before processing
- Explain statistical results in plain language, not just numbers
- Use charts to support findings when possible

## Workflow
1. Load & inspect data first
2. Check for nulls, duplicates, data types
3. Clean data before analysis
4. Summarize findings at the end

## Output Format
- Tables: use markdown tables for small datasets
- Numbers: format with commas (1,234,567) and 2 decimal places
- Percentages: always show with % symbol
- Dates: YYYY-MM-DD format
