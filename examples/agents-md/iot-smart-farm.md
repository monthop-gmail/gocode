# AGENTS.md — IoT Smart Farm

## ระบบ
- ESP32 + Sensors (ความชื้นดิน, อุณหภูมิ, แสง, pH, EC)
- Relay Module ควบคุมปั๊มน้ำ / พัดลม / ไฟ
- MQTT สำหรับสื่อสาร
- LINE Notify สำหรับแจ้งเตือน
- Solar Panel + Battery (off-grid)

## Rules
- ตอบเป็นภาษาไทยเสมอ
- คำนึงถึงการประหยัดพลังงาน (deep sleep, อ่าน sensor ตาม interval)
- ค่า sensor ต้องมี calibration — อธิบายวิธี calibrate ด้วย
- ระวังเรื่อง waterproof สำหรับอุปกรณ์กลางแจ้ง
- ระบบต้อง fail-safe — ถ้า WiFi หลุด ให้ทำงานตาม rule ล่าสุด

## Sensors & Thresholds
| Sensor | ค่าปกติ | ต่ำเกินไป | สูงเกินไป |
|--------|---------|-----------|-----------|
| ความชื้นดิน | 40-70% | < 30% → เปิดน้ำ | > 80% → ปิดน้ำ |
| อุณหภูมิ | 25-35°C | < 20°C → แจ้งเตือน | > 40°C → เปิดพัดลม |
| แสง (LUX) | 10,000-50,000 | < 5,000 → เปิดไฟเสริม | - |
| pH | 5.5-7.0 | < 5.0 → แจ้งเตือน | > 7.5 → แจ้งเตือน |
| EC (mS/cm) | 1.0-2.5 | < 0.5 → เติมปุ๋ย | > 3.0 → เติมน้ำ |

## Automation Rules
```
ถ้า ความชื้นดิน < 30% → เปิดปั๊มน้ำ 5 นาที → ปิด → รอ 10 นาที → วัดอีกครั้ง
ถ้า อุณหภูมิ > 38°C → เปิดพัดลม + แจ้ง LINE
ถ้า อุปกรณ์ offline > 15 นาที → แจ้ง LINE
ถ้า แบตเตอรี่ < 20% → แจ้ง LINE + เพิ่ม sleep interval
```

## Data Logging
- บันทึกค่า sensor ทุก 5 นาที
- สรุปรายวัน: ค่าเฉลี่ย, ค่าต่ำสุด, ค่าสูงสุด
- เก็บ log 90 วัน

## LINE Notify Messages
```
🌡️ อุณหภูมิสูง: 42°C (โรงเรือน A)
💧 เปิดปั๊มน้ำอัตโนมัติ (ความชื้น 25%)
🔋 แบตเตอรี่ต่ำ: Node-003 เหลือ 15%
🔴 อุปกรณ์ offline: Node-005 (20 นาที)
```
